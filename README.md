# ☕ Warm Bean Café – Coffee Ordering Web App

## Overview
Welcome to **Warm Bean Café**, a cozy web app where every cup is crafted with care! ☕  
Place and manage coffee orders easily with our responsive, modern interface.

## Features
- Place coffee orders with size, milk, flavor, and extras
- View all orders in a color-coded table
- Edit or delete orders
- Responsive design for mobile and desktop
- Full-screen hero section with fade-in animation

## Tech Stack
- **Backend:** ASP.NET Core MVC (C#)
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **Database:** SQL Server / Entity Framework Core

## Usage
1. Open the project in **Visual Studio 2022**  
2. Restore NuGet packages  
3. Update your **connection string** in `appsettings.json`
